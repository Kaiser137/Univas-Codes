Desenvolver um programa em Java que manipule nomes de
usuários utilizando vetores, métodos com e sem retorno, e
funções básicas de String. Toda a entrada será feita com
nextLine e a saída com println.

Requisitos Funcionais
O programa deve:
• Cadastro de Nomes:
• Ler e armazenar 5 nomes usando um vetor de String[].
• Utilizar nextLine() para entrada de dados.
• Menu de Operações (com métodos separados):
1. Exibir todos os nomes – método sem retorno.
2. Buscar um nome pelo índice – método com retorno (String).
3. Converter todos os nomes para maiúsculas – método com retorno (String[]).
4. Mostrar o maior nome (mais caracteres) – método com retorno.
5. Contar quantos nomes começam com uma letra específica – método com
retorno (int).
6. Sair


Requisitos Técnicos
• Uso de métodos com e sem retorno.
• Manipulação de vetores e strings (length(), toUpperCase(),
charAt(), etc.).
• Entrada de dados com Scanner.nextLine() e saída com
System.out.println().
• Organização do código em métodos reutilizáveis.
