Exercicios

1. Escreva um programa em C que declare três variáveis: um número inteiro, um número decimal (float) e um caractere. Inicialize essas variáveis com valores e imprima-os usando printf.

2. Crie um programa que leia um número inteiro e um número decimal do teclado e exiba a soma desses dois valores.

3. Escreva um programa que peça ao usuário para digitar um número inteiro. O programa deve verificar se o número é par ou ímpar e exibir a mensagem correspondente.

4. Desenvolva um programa que leia a idade de uma pessoa e verifique se ela é maior de idade (18 anos ou mais) ou menor de idade. Exiba uma mensagem apropriada.

5. Crie um programa que peça ao usuário para digitar dois números inteiros. O programa deve verificar qual número é maior e exibir uma mensagem informando qual é o maior número ou se ambos são iguais.

6. Escreva um programa que leia uma letra do teclado e verifique se é uma vogal ou uma consoante. Exiba a mensagem apropriada.

7. Crie um programa que leia dois números inteiros e exiba o resultado da multiplicação deles.

8. Escreva um programa que leia dois números inteiros e verifique se eles são múltiplos entre si. Exiba uma mensagem indicando se são múltiplos ou não.

9. Desenvolva um programa que leia uma letra, um número inteiro e um número decimal. Em seguida, exiba todos os valores lidos.
